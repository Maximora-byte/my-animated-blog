## 欢迎来到我的博客

这是一个使用 **Next.js** 和 **Tailwind CSS** 构建的现代化博客。

### 特色功能

1. 🎨 **动画登录页** - 可爱的角色会跟随你的鼠标移动
2. 📝 **Markdown 支持** - 使用 Markdown 编写博客文章  
3. 🏷️ **标签系统** - 文章可以添加标签分类
4. 📱 **响应式设计** - 适配各种设备

### 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion

感谢访问！
